Inventory Management System (IMS)
================================

Project Description
------------------
The Inventory Management System for End User Equipment is a web-based application built using Spring Boot MVC framework. It enables organizations to track and manage end-user devices including laptops, desktops, and mobile phones efficiently.

Setup Instructions
------------------
1. Install XAMPP and start MySQL
2. Import database: mysql -u root -p < ims.sql
3. Run the application: ./mvnw spring-boot:run
4. Open browser: http://localhost:8080
5. Login with credentials above

Navigation
----------
- / - Dashboard
- /devices - Device Management
- /employees - Employee Management
- /assignments - Assignment Management
- /issues - Issue Tracking
- /reports - Reports
